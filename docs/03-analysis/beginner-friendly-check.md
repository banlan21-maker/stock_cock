## 4. 개선 권장 사항

### 4.1 반드시 반영 권장 (AI 프롬프트)

1. **공통**
   - 모든 AI 출력에 **"주식 초보자도 이해할 수 있도록 쉬운 말과 친근한 말투로 작성해 주세요. 전문 용어는 피하고, 쓸 때는 꼭 필요한 경우만 쓰고 바로 쉬운 말로 풀어서 설명해 주세요."** 를 추가한다.
2. **뉴스 요약**
   - 요약: "3–4문장으로, **쉬운 말로** 요약해 주세요."
   - 관련 종목 이유: "**한 줄로, 초보자도 이해할 수 있는** 관련 이유를 적어 주세요."
3. **정책 분석**
   - 분석 문단: "**쉬운 말로**, 이 정책이 우리 생활/기업/주가에 어떤 영향을 줄 수 있는지 3–5문장으로 설명해 주세요."
   - 수혜/피해: "**‘누가 득을 보는지’ / ‘누가 부담을 가지는지’** 를 초보자도 이해할 수 있게 한 줄씩 적어 주세요." (수혜주/피해주 용어는 유지하되, 설명은 위와 같이 쉬운 표현 권장)
4. **종목 분석**
   - sentiment: **한글로** "긍정" / "부정" / "중립" 중 하나로 통일하고, 필요 시 "참고할 만함 / 조심할 만함 / 보통" 등 더 친근한 표현 사용 가능.
   - key_factors: "**초보자도 이해할 수 있는 한 줄 요인** (예: ‘반도체 잘 나가서 수혜’, ‘금리 오르면 부담’) 형태로 3개 이내."
   - report: "**전문 용어는 최소화하고, 꼭 쓰면 바로 다음 문장에서 쉬운 말로 풀어서 설명.** 비유나 예시를 넣어도 좋습니다. 5–10문장, 마크다운 형식."

### 4.2 선택 반영 (UI)

- 랜딩/메뉴: "쉽게 정리해 드려요", "한눈에 보는 뉴스/정책/종목" 등 짧은 문구 추가.
- 종목 분석 버튼: "AI가 **쉽게** 풀어서 분석해 드려요" 등으로 변경 검토.
